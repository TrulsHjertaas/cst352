What?

An operating system (OS) is a system software that manages computer hardware and software resources and provides common services for computer programs

Wiki

Why?

An operating system is the most important software that runs on a device. It manages the computer's memory and processes, as well as all of its software and hardware. It also allows you to communicate with the computer without knowing how to speak the computer's language.

GCF-Global

How?

Modern operating systems use a graphical user interface, or GUI. A GUI lets you use your mouse to click icons, buttons, and menus, and everything is clearly displayed on the screen using a combination of graphics and text
